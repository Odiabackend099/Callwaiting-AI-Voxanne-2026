KEY ANIMATION TECHNIQUES USED:

1. BEZIER EASING (Cursor Movement)
   ├─ Formula: (t) => 1 - (1 - t)³
   ├─ Effect: Smooth cubic ease-out
   └─ Used in: Scene 3, 5, 7 (cursor movements)

2. SPRING PHYSICS (Smooth Transitions)
   ├─ Config: { damping: 10 }
   ├─ Effect: Natural acceleration/deceleration
   └─ Used in: Scene 1, 2, 4, 12 (fade-ins and zooms)

3. TYPEWRITER EFFECT (Text Reveal)
   ├─ Formula: Math.floor(interpolate(frame, [start, end], [0, textLength]))
   ├─ Effect: Character-by-character reveal
   └─ Used in: Scene 3, 5, 8 (text input animations)

4. SLIDE TRANSITION (Scene Change)
   ├─ Formula: interpolate(frame, [0, 60], [0, -100]) for exit
   ├─ Effect: Smooth wipe between scenes
   └─ Used in: Scene 6 (most visible, most complex)

5. STAGGERED ANIMATIONS (Visual Rhythm)
   ├─ Pattern: Card N uses frames [15*N, 15*N + 30]
   ├─ Effect: Sequential reveals for visual interest
   └─ Used in: Scene 10 (slides), Scene 11 (fades)

6. PULSING ANIMATION (Attention)
   ├─ Formula: 1 + 0.1*sin((frame/fps)*PI*2)
   ├─ Effect: Continuous oscillation using sine wave
   └─ Used in: Scene 6 (code highlight)
