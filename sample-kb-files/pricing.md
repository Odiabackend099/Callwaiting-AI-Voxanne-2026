# Voxanne AI - Pricing Guide

## Service Tiers

### Essentials Plan
- **Price**: £169/month + setup fee
- **Features**:
  - 24/7 AI receptionist
  - Unlimited inbound calls
  - Basic lead capture
  - Email notifications
  - Standard support

### Growth Plan
- **Price**: £289/month + setup fee
- **Features**:
  - Everything in Essentials
  - Advanced lead qualification
  - SMS notifications
  - Priority support
  - Custom call scripts
  - Integration with booking systems

### Premium Plan
- **Price**: £449/month + setup fee
- **Features**:
  - Everything in Growth
  - Dedicated account manager
  - Custom AI training
  - Advanced analytics
  - API access
  - White-label options
  - 24/7 phone support
